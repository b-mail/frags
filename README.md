<h2>FRAG?</h2>
<p>
FRAG(프랙)은 FRAGS를 구성하는 요소입니다. FRAG은 취미나 관심사가
같은 사람들, 함께 공부하는 사람들, 친한 친구들끼리 모이는 공간이 될
수 있습니다.
</p>
<p>
지금 바로 여러분이 원하는 FRAG을 만들어 여러분의 친구들을
초대해보세요!
</p>
